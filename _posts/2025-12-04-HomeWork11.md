# Wiener Process Simulation (Brownian Motion)

## 1. Objective

The goal of this homework is to simulate a **Wiener process** (also known as **Brownian motion**) using the **Euler–Maruyama method**, previously developed for stochastic differential equations (SDEs).

Optionally, the simulator can be generalized to handle more general SDEs of the form:

$$
dX_t = \mu(X_t, t)\,dt + \sigma(X_t, t)\,dW_t
$$

---

## 2. Wiener Process Definition

A Wiener process $\( \{W(t)\}_{t \ge 0} \)$ is a continuous-time stochastic process satisfying:

- $\( W(0) = 0 \)$
- Independent increments
- Stationary increments
- Continuous sample paths
- Normally distributed increments:
  $$
  W(t+\Delta t) - W(t) \sim \mathcal{N}(0, \Delta t)
  $$

The Wiener process is the fundamental building block of stochastic calculus.

---

## 3. Stochastic Differential Equation Formulation

The Wiener process satisfies the SDE:

$$
dW(t) = dB(t)
$$

where $\( B(t) \)$ is standard Brownian motion.

---

## 4. Euler–Maruyama Discretization

To simulate the process numerically, the time interval $\( [0,T] \)$ is divided into $\( n \)$ steps of size:

$$
\Delta t = \frac{T}{n}
$$

The Euler–Maruyama update rule becomes:

$$
W_{k+1} = W_k + \sqrt{\Delta t}\, Z_k
$$

where:
- $\( Z_k \sim \mathcal{N}(0,1) \)$ are independent standard normal variables.

let wienerChart = null;

function simulateWiener() {
  const n = parseInt(document.getElementById("steps").value);
  const T = parseFloat(document.getElementById("T").value);
  const dt = T / n;

  let t = [];
  let W = [];
  let value = 0;

  for (let i = 0; i <= n; i++) {
    t.push(i * dt);
    W.push(value);
    value += Math.sqrt(dt) * gaussian();
  }

  if (wienerChart) wienerChart.destroy();

  wienerChart = new Chart(document.getElementById("wienerChart"), {
    type: "line",
    data: {
      labels: t,
      datasets: [{
        label: "Wiener Process W(t)",
        data: W,
        borderWidth: 2,
        pointRadius: 0
      }]
    },
    options: {
      scales: {
        x: { title: { display: true, text: "Time t" }},
        y: { title: { display: true, text: "W(t)" }}
      }
    }
  });
}

/* Standard normal generator (Box–Muller) */
function gaussian() {
  let u = 0, v = 0;
  while (u === 0) u = Math.random();
  while (v === 0) v = Math.random();
  return Math.sqrt(-2.0 * Math.log(u)) * Math.cos(2.0 * Math.PI * v);
}
</script>

## 6. Optional Generalization to SDEs

A general stochastic differential equation has the form:

$$
dX_t = \mu(X_t,t)\,dt + \sigma(X_t,t)\,dW_t
$$

The Euler–Maruyama scheme becomes:

$$
X_{k+1} = X_k + \mu(X_k,t_k)\Delta t + \sigma(X_k,t_k)\sqrt{\Delta t}\,Z_k
$$

This allows simulation of:
- Geometric Brownian motion
- Ornstein–Uhlenbeck processes
- Financial and physical diffusion models

---

## 7. Interpretation

- The simulated paths are **continuous but nowhere differentiable**
- Variance grows linearly with time:
  $$
  \text{Var}(W(t)) = t
  $$
- Different runs produce different trajectories, but identical statistical properties

---

## 8. Conclusion

The Euler–Maruyama method provides a simple and effective way to simulate the Wiener process. This approach illustrates how continuous-time stochastic behavior emerges from discrete random increments and serves as the foundation for more complex stochastic differential equation models.
